# CSC-289 Repository

## 📘 Overview

This repository contains coursework, exercises, and supporting materials for **CSC-289**. It is structured to guide students through foundational concepts, hands-on workflows, and practical assignments designed to build technical skills step by step.

The repository is organized into modules, each focusing on specific learning objectives and activities.

---

## 📂 Repository Structure

```
CSC-289-main/
│
├── README.md                     # Main project documentation
├── verify_environment.py        # Script to check environment setup
│
└── module1/
    ├── README.md                # Module 1 overview
    ├── DAY1_GET_UNBLOCKED.md    # Setup and troubleshooting guide
    ├── WEEK1_WORKFLOW_BOOTCAMP.md # Workflow training materials
    ├── SACRED_FLOW_REFERENCE.txt # Reference guide
    ├── t_1_1_submission.md      # Assignment submission
    └── verify_environment.py    # Module-specific environment check
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd CSC-289-main
```

### 2. Verify Your Environment

Run the environment check script to ensure everything is set up correctly:

```bash
python verify_environment.py
```

If working within a module:

```bash
cd module1
python verify_environment.py
```
